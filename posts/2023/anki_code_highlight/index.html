Anki_code_highlight

January 9, 2023 · 1 min | Suggest Changes

Recently I’ve decided to use Anki to track my leetcode progress.

I’m following the advice from reddit.

Personally once I’ve successfully submitted a problem, I’d like to save it as a Anki card. Anki do its best to regularly remind me to refresh.

However it turns out Anki doesn’t support code highlighting out of box, and the top result from Google (Syntax Highlighting for Code) is a little bit outdated. So I decided to write a single page app with hightlightjs to get the job done.
